# Placeholder Images

This directory should contain your wedding photos. For the demo, use placeholder images from:
- https://picsum.photos/ (random photos)
- https://placeholder.com/ (simple placeholders)
- https://unsplash.com/ (free stock photos)

Required images:
- Hero.JPG (main banner)
- bride.jpeg (bride photo)
- groom.jpeg (groom photo)
- 1.jpeg through 10.jpeg (memory carousel)
- timeline1.png, timeline2.jpeg, Propose.jpeg, big-day.jpeg (timeline)
- Family photos: bride-parents.jpg, groom-parents.jpeg, etc.
- {{Groom Surname}}.jpeg, {{Bride Surname}}.jpg (family group photos)

Recommended size: 1920x1080 for hero, 800x800 for portraits
